# LDE — Legitimacy Dynamics Engine (Component 5)

Calibration code and data for Preprint #4:
Serban, G.F. (2026). Legitimacy Dynamics Engine: Cross-national empirical
calibration on Standard Eurobarometer data (2014–2025). SSRN Working Paper.

## Files
- `lde_calibration.py` — main calibration script (differential evolution)
- `data_eurobarometer_lde.csv` — L(t) series for Greece, Hungary, Serbia
- `bootstrap_results.csv` — 500 bootstrap R² replications per country

## Requirements
Python 3.x, scipy, numpy, pandas
